# Dashboard sync

Read `docs/workflow/project/dev-dashboard.md`. Preserve the authority split:

- Human judgment: `docs/workflow/project/dashboard-state.json`.
- Machine facts: generated by `scripts/dashboard/generate-state.mjs`.
- Role/Skill truth: `.agents/registry.json` and `.codex/skills/`.
- Feature state: `docs/workflow/features/<slug>/dev_log.md`.

Run:

```bash
npm run workflow:generate
npm run dashboard
npm run dashboard:verify
```

Report branch, commit, dirty files, current phase, missing required documents,
runtime mirror drift, and verification result. Fix deterministic generator or
mirror drift when authorized. Never change priority, accept risk, create a
branch, mark Ship, or convert a failed/blocked check into pass.
